## üé® Logo System

### Logo Versions

#### Icon Version
- **File**: `icon-enhanced.svg`
- **Usage**: Sidebar, favicon, small displays
- **Design**: Goose with compass elements and impact waves
- **Colors**: Deep Teal body, Soft Coral beak and legs

#### Full Logo
- **File**: `logo-enhanced.svg`
- **Usage**: Header, hero sections
- **Design**: Icon + "NavImpact" wordmark
- **Typography**: Carrotflower for wordmark

### Logo Guidelines

- **Minimum Size**: 24px for icon, 120px width for full logo
- **Clear Space**: Equal to the height of the "N" in NavImpact
- **Background**: Works on white, light gray, and gradient backgrounds
- **Don't**: Stretch, rotate, add effects, or change colors

## üì± Responsive Design

### Breakpoints

| Device | Width | Layout | Navigation |
|--------|-------|--------|------------|
| Mobile | ‚â§ 768px | Single column | Stacked |
| Tablet | 768px - 1024px | Two columns | Horizontal |
| Desktop | ‚â• 1024px | Multi-column | Full |

### Mobile Considerations

- **Touch Targets**: Minimum 44px height/width
- **Font Sizes**: Reduced by 20% for mobile
- **Spacing**: Tighter spacing for mobile
- **Navigation**: Collapsible menu for mobile

## üé≠ Motion & Interactions

### Micro-interactions

- **Hover Duration**: 250ms ease-in-out
- **Button Hover**: 1px lift + shadow increase
- **Card Hover**: 2px lift + shadow increase
- **Logo Hover**: Subtle rotation and scale

### Animations

- **Fade In**: 600ms ease-out
- **Slide In**: 400ms ease-out
- **Float**: 3s ease-in-out infinite

### Accessibility

- **Reduced Motion**: Respects `prefers-reduced-motion`
- **Focus States**: 2px Soft Coral outline
- **High Contrast**: Enhanced colors for accessibility
